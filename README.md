# theflat.gen.nz

Static site content for **theflat.gen.nz**, hosted on GitHub Pages.

## What's in here

- `index.html`: site homepage
- `robots.txt`: crawler rules
- `sitemap.xml`: sitemap for search engines
- `llms.txt`: guidance for LLM tooling
- `site.webmanifest` + icons: installable site metadata (PWA-style)
- `404.html`: not-found page (used by GitHub Pages)
- `.well-known/security.txt`: security contact information
- `.nojekyll`: disables Jekyll processing on GitHub Pages

## Notes (GitHub Pages)

- If you use a custom domain, GitHub Pages typically manages the `CNAME` for you via repo settings; alternatively you can add a `CNAME` file at the repo root.
- Update `sitemap.xml` when you add new pages.
